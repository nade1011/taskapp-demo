# TaskApp (Beta)

TaskAppは、シンプルなタスク管理用のWebアプリです。
1つのプロジェクトに集中し、進行状態を一画面で把握できる構成になっています。

## Features
- タスク／セクション管理
- 状態管理（未着手・進行中・完了）
- プロジェクト単位でのArchiveエクスポート／インポート
- LocalStorageによるデータ保存
- 一画面で全体を把握できるUI
