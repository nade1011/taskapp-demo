# TaskApp (Beta)

TaskAppは、単にタスクを消化するためのツールではなく、  
**作業を伴うすべての人のプロジェクト進行を補佐する**ことを目的とした  
シンプルなタスク管理アプリです。

制作・思考・判断を伴う作業では、  
「何をやるか」よりも  
「どこまで進んでいるか」「次に何をするか」  
がすぐに分かることが重要だと考えています。

本アプリでは、状態の分かりやすさと再開のしやすさを重視し、  
認知負荷をできるだけ減らす設計を行っています。

---

## 概要

TaskAppは、1つのプロジェクトに集中することを前提とし、  
プロジェクト進行に必要な情報を  
**一つの画面に集約して把握できる**設計を採用した  
タスク管理アプリです。

複数の画面を行き来することなく、  
画面を開いた瞬間にプロジェクト全体の状態を把握できる  
シンプルな構成を採用しています。

---

## アプリの目的

既存のタスク管理ツールでは、

- 機能や設定が多く、状態が分かりづらい  
- 作業を翌日に引き継ぐ際、どこまで進めていたか分からない  
- 管理そのものが思考の妨げになる  

といった課題を感じていました。

そこで本アプリでは、  
**「管理のために考えさせない」**ことを重視し、  
タスク管理がプロジェクト進行の妨げにならないことを目的としています。

---

## 主な機能

- タスク／セクション管理
- 状態管理（未着手・進行中・完了）
- プロジェクト単位でのArchiveエクスポート
- Archiveからのインポート機能
- LocalStorageによるデータ保存
- 情報を一画面で把握できるUI構成

---

## UI / デザイン方針

- 見た目よりも **読みやすさ・分かりやすさを優先**
- 状態が一目で把握できる情報設計
- ガイドページの左寄せレイアウトによる安定した可読性
- 不要な装飾や操作を省いたシンプルなUI
- 学習コストを発生させない設計
  

---

## データ設計 / アーキテクチャ

- データはLocalStorageで管理し、外部サービスに依存しない構成
- プロジェクトはArchive機能により退避・再利用が可能
- 「削除」ではなく「区切り」を意識した運用設計

---

## 既知の課題 / 今後の改善

- よく使う構成を素早く呼び出せるテンプレート機能の追加
